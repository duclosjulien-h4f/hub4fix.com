# Hub4Fix — Landing Page

Page de présentation du projet **Hub4Fix**, plateforme française d'économie circulaire spécialisée dans l'impression 3D de pièces détachées orphelines.

**Philosophie :** Le Kintsugi Numérique — transposer l'art japonais de la réparation sublimée à l'impression 3D. Là où le Kintsugi utilise l'or, Hub4Fix signe ses pièces en rouge.
